# TJ-ChaseVoice
猫和老鼠手游语音数据集

## 说明
本数据集在Eternity-Ixy的[原数据集](https://github.com/Eternity-Ixy/TJ-ChaseVoice)的基础上进行了标注，标注文件位于每个角色文件夹的 `text.txt`

标注格式：

    文件名|标注内容

注：标注内容按文件名的名称升序排列

此外，对同一角色的不同配音做了较严格区分，请自行判断是否混合

此外，将无实义的音频放入角色文件夹下的 `mood` 文件夹中，唱歌的音频则放入 `sing` 文件夹中
